# sample-api-mock
api mock sample (for API Blueprint)

### Get the code
```
git clone https://github.com/nishi2244/sample-api-mock
cd sample-api-mock
```

### Install dependencies
```
npm install
```

### Run it
terminal command
```
npm run drakov
```

### Stop
Keyboard shortcut. crtl + c.

### Modify api
Edit markdown files.
```
vi ./src/message.md
```
